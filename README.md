# 💸 Bill Splitter

A Python command-line tool that calculates how much each person owes 
after splitting a restaurant bill, including tip.

## Features
- Splits bill among any number of friends
- Calculates tip automatically
- Displays a clean, formatted bill summary

## How to Run
```bash
python bill_splitter.py
```
